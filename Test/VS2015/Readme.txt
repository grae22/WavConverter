Building the VS2015 solution
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

x Add the following environment vars:
  x BoostInclude - path to the Boost include folder, e.g. c:\boost\boost_1_61_0\
  x BoostLibs - path to the Boost libs folder, e.g. c:\boost\boost_1_61_0\libs\
  x GTestInclude - path to GoogleTest include folder, e.g. c:\gtest\include\
  x GTestLibsDebug - path to GoogleTest debug libs folder, e.g. c:\gtest\debug\
  x GTestLibsRelease - path to GoogleTest debug libs folder, e.g. c:\gtest\release\
  x GTestLibsDebug_x64 - path to GoogleTest debug libs folder, e.g. c:\gtest\x64\debug\
  x GTestLibsRelease_x64 - path to GoogleTest debug libs folder, e.g. c:\gtest\x64\release\
